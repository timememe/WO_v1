Strudel - это язык для live coding музыки в браузере. Используй мини-нотацию для создания паттернов:
• sound("bd sd") - базовый драм-паттерн
• .fast(2) - ускорить в 2 раза
• .slow(0.5) - замедлить
• stack(...) - наложить паттерны
